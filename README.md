# Pie Ray-dle

This site is a simple site intended to teach and allow you to practice the "pie ray" or "piedar" or "pie charting" strategy used in Minecraft speedrunning.

It introduces the basic approaches over 4 stages which allow you to build up base concepts towards this skill, and then repeatedly practice the technique.

Visit [Pie ray-dle](https://nick-hiebl.github.io/pie-raydle/).
